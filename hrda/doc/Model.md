모형작업
================

## 1\. 모형

전처리 완료한 인사데이터를 활용해 모형을 만들고 이를 머신 러닝으로 구축할 준비를 한다.

## R Source Code

You can include R code in the document as follows:

``` r
str(data)
```

## Including Plots

You can also embed plots, for example:

``` r
plot(pressure)
```
